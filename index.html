手机 / App 端打开

这是网页，不是原生 App。推荐两种方式：

  • 同一 WiFi：电脑终端运行 python3 -m http.server 8765（在 gap-plan-app 目录），手机 Safari 打开 http://电脑IP:8765 → 分享 → 添加到主屏幕
  • 在线地址：把文件夹部署到 Vercel / Netlify / GitHub Pages，手机用固定 URL 打开并加主屏幕

Firestore 云同步

Google 登录后，手机与电脑自动同步同一份数据（按 _updatedAt 取较新）。Export/Import 仍可作为备份。

未连接

iPhone 注意：请用 Safari 地址栏打开 gap-plan-app.vercel.app#sync 登录，不要从「主屏幕图标」里点 Google 登录（会失败）。登录成功后再加主屏幕。

首次使用：控制台开启 Google 登录 + 发布 Firestore 规则 → 两端用同一 Google 账号登录。
